# League Companion MVP

A lightweight League of Legends companion website built with React and Tailwind CSS.

## Features
- Champion, role, and enemy composition selection
- Rune setup recommendations
- Item build suggestions with situational adaptations
- In-depth strategy analysis
- Responsive dark theme layout
- Live Riot Data Dragon champion and item metadata lookup

## Run locally
1. Install dependencies
   ```bash
   npm install
   ```
2. Start the development server
   ```bash
   npm run dev
   ```
3. Open the local URL shown in the terminal

## Notes
- The current MVP fetches real champion and item metadata from Riot Data Dragon.
- Recommendation logic uses champion tags and role profiles to generate rune/item suggestions.
- The app is built to support future Riot API or AI enhancements.
